# ⚡ Power Load Forecasting using Deep MLP with Multivariate Meteorological Features

An end-to-end Machine Learning application that predicts electrical power load using a Deep Multi-Layer Perceptron (MLP) model trained on multivariate meteorological data. The project includes data preprocessing, model training, evaluation, and deployment through a Flask web application for real-time forecasting.

---

## 📖 Overview

Accurate power load forecasting is essential for efficient power generation, grid stability, and energy management. This project leverages Deep Learning to forecast electricity demand using weather conditions and calendar-based features.

The trained model is deployed as a user-friendly Flask web application, allowing users to input meteorological parameters and receive instant power load predictions.

---

## 🎯 Objectives

- Forecast electrical power load using Deep Learning.
- Analyze the influence of meteorological variables on energy demand.
- Improve forecasting accuracy through feature scaling and neural networks.
- Deploy the trained model using Flask for real-time predictions.
